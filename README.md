# Automated Email Campaign Portal

A long-term automated workflow solution powered by n8n backend automation for scalable and paperless email campaign management.

---

## Overview

This platform was developed to simplify and automate promotional email campaigns using a professional frontend interface integrated with an n8n backend workflow.

The system allows users to:

- Upload CSV/XLS/XLSX recipient lists
- Send automated email campaigns
- Attach optional files
- Use HTML-formatted email content
- Process multiple recipients automatically
- Run scalable workflow automation

## Workflow Logic

The n8n workflow includes:

1. Form Trigger
2. Recipient Extraction
3. Recipient Splitting
4. Email Data Preparation
5. Automated Signature Integration
6. Batch Processing
7. Gmail Email Sending
8. Campaign Summary Generation
